# 詐騙案例訓練數據：釣魚簡訊詐騙

## 詐騙類型定義
「釣魚簡訊詐騙」是一種透過偽裝成官方機構或知名企業的訊息，誘導使用者點擊惡意連結的詐騙手法。詐騙者常利用緊急通知（如帳戶異常、罰單未繳、包裹配送失敗等）引起恐慌，或以利益誘惑（如退稅通知、中獎訊息等）引誘受害者點擊釣魚網站連結，進而騙取個人資料、帳號密碼或金融資訊。這類詐騙常透過簡訊、電子郵件或社群媒體私訊進行，並經常使用縮短網址或模仿官方網址，增加辨識難度。

## 特徵標記

### 1. 常見話術標記
以下為詐騙者常使用的話術，AI可藉以辨識潛在詐騙：

```
<詐騙話術>
- 銀行帳戶因異常操作已被暫時凍結
- 包裹配送失敗需重新確認地址
- 財政部退稅金額尚未領取
- 訂單因付款異常被暫停處理
- 社群帳號異常登入需確認身份
- 系統升級需完成更新
- 貸款申請已批准需確認信息
- 話費尚有欠款將暫停服務
- 醫療檢測報告已出可查看
- 逾期罰單通知需立即處理
- 信用卡異常交易需確認
- 政府補助金需登記領取
- 中獎通知需驗證身份
- 安全認證已過期需更新
- 會員積分即將到期
- 稅務問題需立即處理
- 驗證碼已發送請確認
</詐騙話術>
```

### 2. 常見誘導行為標記
詐騙過程中常見的誘導行為：

```
<誘導行為>
- 製造緊急感和時間壓力（「立即處理」、「24小時內」）
- 使用威脅性語言（「否則帳戶將被永久凍結」）
- 提供縮短的網址連結（如bit.ly、tinyurl等）
- 模仿官方網址但有細微差異（bankoftaiwan.tw vs banktaiwan-tw.com）
- 使用非官方的發信地址（如gmail、yahoo等免費郵箱）
- 強調必須通過提供的連結處理問題
- 聲稱獲得意外收益（退稅、中獎等）
- 利用當下熱門話題（如疫情補助、災害救助金）
- 使用偽造的官方標誌或視覺設計
- 信件或訊息中存在拼寫錯誤或語法不自然
- 要求輸入個人敏感資訊（身份證號、銀行帳號等）
- 聲稱來自權威機構增加可信度
</誘導行為>
```

### 3. 詐騙流程模式
完整的詐騙流程模式：

```
<詐騙流程>
1. 偽裝成官方機構或知名企業發送訊息
2. 利用緊急情況或利益引誘點擊連結
3. 提供不明或縮短的網址連結
4. 受害者點擊連結後進入釣魚網站
5. 釣魚網站模仿官方網站外觀，增加可信度
6. 誘導輸入個人資料、帳號密碼或金融資訊
7. 收集資料後可能顯示錯誤訊息或跳轉頁面
8. 詐騙者取得資料後進行盜用、盜刷或身份盜用
9. 受害者通常在發現異常交易或帳戶活動後才察覺被騙
</詐騙流程>
```

## 案例分析

### 案例一：逾期罰單釣魚詐騙

```
<案例描述>
受害者收到一封看似來自監理站的電子郵件，信件主旨為「逾期罰單通知」，發件人顯示為「guy.toelen1@telenet.be」。信件內容指出受害者有一張罰單尚未繳納，強調如不立即處理，將衍生許多不必要的法律問題。

郵件中附有一個網址連結「www.kvrplmj.cc」，聲稱點擊後可以快速繳費。受害者認為既然是監理站的通知信件，應該可信，於是點擊連結進入一個看似官方的繳費網站，並依照指示輸入信用卡資料完成「繳款」。

繳款過程中，網站顯示「繳款失敗，請稍後重試」的訊息，受害者當時沒有特別在意。然而，隔天查閱信用卡帳單時，發現被莫名扣除3萬多元及489元手續費。受害者才意識到可能遭遇詐騙，立即報警處理。

事後了解，這是詐騙集團精心設計的釣魚網站，他們冒充監理站發送假通知，利用「逾期罰單」的恐懼心理誘使受害者點擊連結並提供金融資訊，最終導致信用卡被盜刷。
</案例描述>
```

### 關鍵識別點
AI應學會識別以下關鍵點：

```
<關鍵識別點>
1. 發信地址不是官方網域（非gov.tw結尾）
2. 不尋常的發件人地址（guy.toelen1@telenet.be）
3. 提供的網址可疑（www.kvrplmj.cc非官方網域）
4. 未具體說明罰單詳情（如違規事項、車牌號碼等）
5. 強調後果嚴重製造恐慌
6. 催促立即通過連結處理
7. 繳費過程出現異常（顯示繳款失敗）
8. 要求提供完整信用卡資料
9. 不提供其他官方繳費管道選擇
10. 語言表達可能不夠專業或有錯誤
</關鍵識別點>
```

## 防詐教育重點

### 防詐技巧
AI應提供以下防詐建議：

```
<防詐技巧>
1. 監理機構絕不會以電郵或不明網址傳送罰單
2. 收到疑似監理機構催繳電郵，不要理會，並直接向監理機構查證
3. 不要點擊不明連結、不要輸入個人或信用卡資訊
4. 檢查發信地址是否為官方網域(.gov.tw等)
5. 官方機構不會使用免費郵箱或不明網域
6. 懷疑訊息真實性時，直接聯繫官方客服查證
7. 使用官方APP或直接輸入官網網址，而非通過連結
8. 啟用金融卡或信用卡的交易通知服務
9. 定期檢查銀行和信用卡對帳單
10. 已點擊可疑連結應立即更改密碼並通知相關機構
11. 對於要求個人或金融資訊的網站保持警惕
12. 留意網址拼寫是否正確（詐騙網站常有微小差異）
</防詐技巧>
```

### 警示信號
AI應辨識以下警示信號：

```
<警示信號>
- 來自陌生或可疑的寄件人
- 使用不明或縮短的網址
- 強調緊急性和嚴重後果
- 存在拼寫錯誤或語法不自然
- 要求立即採取行動
- 提供的網址與官方網址不符
- 要求提供敏感個人資料
- 電子郵件地址與官方網域不符
- 使用威脅或恐嚇語氣
- 承諾不合理的利益或獎勵
- 未具名或使用通用稱謂
- 訊息中缺乏具體細節
- 多個錯別字或格式混亂
- 不提供其他聯繫或查證方式
</警示信號>
```

## 模擬對話示例
以下是詐騙對話示例，用於AI訓練：

```
<詐騙對話>
詐騙簡訊: 【中華郵政】您的包裹因地址不完整無法配送，請於24小時內點擊連結更新資料，否則將退回寄件人：http://bit.ly/2xyzABC
受害者訊問: 請問這個連結是什麼？我最近有在等包裹。

AI回應: 這是詐騙簡訊！請勿點擊連結！中華郵政不會使用短網址(bit.ly)通知包裹問題。若包裹配送失敗，郵差會留下紙本通知單，或直接撥打您的聯絡電話。如您確實在等待郵件，請直接撥打中華郵政官方客服電話查詢。這類釣魚簡訊目的是誘導您點擊惡意連結，竊取個人資料。請立即刪除此訊息。

詐騙簡訊: 【台灣銀行】您的網路銀行帳戶安全驗證已過期，為避免帳戶被凍結，請立即點擊更新：https://tw-bank-verify.com
受害者訊問: 我有台銀帳戶，這是真的嗎？需要更新嗎？

AI回應: 這是詐騙簡訊！請勿點擊！台灣銀行不會透過簡訊要求您點擊連結更新安全驗證。正確的網址應為www.bot.com.tw，而非tw-bank-verify.com。這是典型的釣魚網站，目的是騙取您的帳號密碼。若您擔心帳戶安全，請直接開啟台銀官方APP或撥打銀行客服電話查詢。切勿點擊簡訊中的連結，以保護您的帳戶安全。

詐騙郵件: 主旨：【財政部】111年度所得稅退稅通知
親愛的納稅人您好，
財政部國稅局通知您有NT$8,742元的退稅款項尚未領取，請於7日內點擊以下連結填寫帳戶資料，以利退款作業：
https://finance-tax-return.com/tw/refund
逾期未處理，退稅金額將予以沒收。
財政部國稅局 敬上
受害者訊問: 我有報稅，可能真的有退稅，要點進去填寫嗎？

AI回應: 這是詐騙郵件！請勿點擊連結或提供任何資訊！財政部不會用這種方式通知退稅，也不會威脅「退稅金額將予以沒收」。合法的退稅會直接匯入您報稅時提供的帳戶或寄發退稅支票，無需點擊任何連結。此外，官方網址應為www.etax.nat.gov.tw，而非finance-tax-return.com。這是釣魚網站，目的是竊取您的個人及銀行資料。如要查詢退稅情況，請直接登入財政部稅務入口網或洽詢各地國稅局。
</詐騙對話>
```

## AI回應策略
當用戶諮詢類似詐騙情況時，AI應：

1. 立即識別「釣魚簡訊詐騙」的特徵
2. 明確警告不要點擊任何不明連結
3. 說明為何這是詐騙（如非官方網域、要求敏感資訊等）
4. 解釋正規機構或企業的實際通知方式
5. 提供正確的查證管道（如官方網站、客服電話）
6. 建議若已點擊連結應採取的措施（更改密碼、通知銀行等）
7. 提醒用戶檢查發件人地址和網址的真實性
8. 解釋詐騙者如何利用恐慌或利益誘惑進行詐騙
9. 鼓勵向165反詐騙專線舉報
10. 強調保護個人和金融資訊的重要性
